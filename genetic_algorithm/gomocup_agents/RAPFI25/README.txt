This is the single-threaded version of Rapfi, built for Gomocup 2025.

For machines that support newer instruction sets, the order of preference is:

AVX512VNNI > AVX512 > AVXVNNI > AVX2

Please select the first one that can be run on the target machine.

To find the multi-threaded version of Rapfi together with a user interface,
please visit Rapfi's Github repo: https://github.com/dhbloo/rapfi
